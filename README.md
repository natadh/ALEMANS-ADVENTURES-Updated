# ALEMANS-ADVENTURES - East African Safari Platform

A live travel booking platform for East African safari destinations, covering national parks, wildlife experiences, culture and beach & water sports across Kenya, Uganda, Tanzania, Rwanda and Zanzibar.

🌍 **[View live site](https://alemansadventures.com/)**

---

## My contributions

This is a collaborative project. I was responsible for:

- **Contact page** — Implementation from layout to form structure
- **Journals & reviews page** — Initial architecture and component setup (later extended with backend integration by a colleague)
- **Safari destination data** — authored JSON files for all safari destinations, including national park structure and destination details for Rwanda and Zanzibar
- **Photo integration** — added photography assets across destination pages

---

## Tech stack

- React · TypeScript · Vite
- JSON-based content structure
- REST API integration (backend by collaborator)

---

## What I learned

This was my first production React project. The biggest learning curve was component architecture — understanding how to structure reusable components, manage props across a tree, and keep state in the right place. Working on a live codebase with a collaborator also meant learning to read and build on someone else's code, not just my own.
